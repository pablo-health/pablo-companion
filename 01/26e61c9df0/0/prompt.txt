## Task 2: GitHub Actions Workflows (matching AudioCaptureKit)

### `.github/workflows/ci.yml` — Main CI
- **Triggers**: push to `main`, PR to `main`, `workflow_dispatch`
- **Concurrency**: cancel in-progress on PR
- **Jobs**:
  1. **detect-changes** — `dorny/paths-filter` to detect `mac/**` and `core/**` changes
  2. **swift-lint** — macOS 15, SwiftLint 0.58.0 + SwiftFormat 0.55.3 on `mac/`
  3. **swift-build** — `xcodebuild build` on `mac/` (conditional: mac changes or manual)
  4. **rust-lint** — rustfmt + clippy on `core/` (conditional: `core/Cargo.toml` exists AND core changes)
  5. **rust-test** — `cargo test` on `core/` (same condition)
  6. **dependency-review** — PRs only; deny HIGH/CRITICAL, deny GPL-2/3 and AGPL-3

Note: Swift test job omitted for now (no tests yet); add when `make test-mac` is ready.

### `.github/workflows/codeql.yml` — CodeQL Security
- **Triggers**: push to `main`, PR to `main`, weekly (Wednesday 8AM UTC)
- **Jobs**: analyze Swift (`mac/`) + analyze JS if any web assets added later

### `.github/workflows/security.yml` — Trivy Scanning
- **Triggers**: push to `main`, PR to `main`, weekly (Monday 6AM UTC), `workflow_dispatch`
- **Jobs**: filesystem vulnerability scan (HIGH/CRITICAL) + config/secrets scan
- Upload results as SARIF to GitHub Security tab

### `.github/workflows/rust-security.yml` — Rust Security
- **Triggers**: push to `main`, PR to `main`, weekly (Tuesday 7AM UTC), `workflow_dispatch`
- **Condition**: only runs if `core/Cargo.toml` exists
- **Jobs**: cargo-audit + cargo-deny

### `.github/workflows/scorecard.yml` — OpenSSF Scorecard
- **Triggers**: push to `main`, weekly (Thursday 9AM UTC), `workflow_dispatch`
- Publishes SARIF to Security tab

### `.github/dependabot.yml`

## Task 3: Makefile

Referenced throughout CLAUDE.md; mirrors what AudioCaptureKit CI calls:

```makefile
.PHONY: check build-mac test-mac build-core test-core lint-swift

check: lint-swift build-mac build-core test-core

lint-swift:
	swiftlint --strict mac/
	swiftformat --lint mac/

build-mac:
	xcodebuild -project mac/PabloCompanion.xcodeproj \
	  -scheme PabloCompanion \
	  -destination 'platform=macOS' \
	  build | xcpretty

test-mac:
	xcodebuild -project mac/PabloCompanion.xcodeproj \
	  -scheme PabloCompanion \
	  -destination 'platform=macOS' \
	  test | xcpretty

build-core:
	@if [ -f core/Cargo.toml ]; then cargo build --manifest-path core/Cargo.toml; fi

test-core:
	@if [ -f core/Cargo.toml ]; then cargo test --manifest-path core/Cargo.toml; fi
```
