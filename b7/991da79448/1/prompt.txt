## Step 6: Font System

**Create `mac/PabloCompanion/Extensions/Font+Pablo.swift`**

```swift
import SwiftUI

extension Font {
    // Display headings — Fraunces if bundled, warm serif fallback
    static func pabloDisplay(_ size: CGFloat) -> Font {
        Font.custom("Fraunces", size: size)
    }
    // Body and labels — DM Sans if bundled, clean system fallback
    static func pabloBody(_ size: CGFloat) -> Font {
        Font.custom("DM Sans", size: size)
    }
}
```

`Font.custom` silently falls back to system font if the font isn't bundled — so this is safe to ship now. Font files (DM Sans, Fraunces from Google Fonts) can be added in a follow-up task without changing any call sites.
